Introduction

GDB is an incredibly powerful C/C++ debugging tool. Learn to use it at its full potential - it'll save you lots of time and tears over your programming career.

This is the companion website to the talk given on Sept. 28, 2014 at The George Washington University. The talk was given by Phil Lopreiato and Neel Shah. The repository contains all of the code used to accompany the talk, as well as some helpful documents. Our slides are embedded below.
